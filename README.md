# PPT_AddIn

PowerPointの図形をドラッグ＆ドロップで移動させるアドインの開発・配布用リポジトリです。

## 開発履歴

### PowerPointオブジェクトドラッグ機能 (C# VSTO版)
- **内容**: スライドショー中に特定の図形をマウス・タッチで自由に移動させるアドイン。
- **成果**: VBA版の不安定さを解消し、Win32 API（マウスフック・タッチフック）とC#を用いた高度な制御により、滑らかで安定した動作を実現しました。
- **主な機能**:
    - **図形名プレフィックス**: 図形名を `Drag_` で始めるだけでドラッグ対象に設定。
    - **マウス・タッチ対応**: マウスドラッグで図形を移動可能（タッチドラッグは現在安定性の向上のため無効化中）。
    - **自動位置リセット**: スライドショー終了時に、動かした図形が自動的に元の位置に戻ります。
    - **ペン描画パレット**: スライド上にペン・蛍光ペン・消しゴムで書き込み可能。スライドごとに描画を保持。
    - **スライドナビゲーション**: パレットのボタンでスライド送り/戻しが可能。

## 導入・利用ガイド

### インストール方法 (ClickOnce)
1.  Visual Studio で `PPTDragDropAddIn` プロジェクトを右クリックし、**「発行 (Publish)」** を実行します。
2.  生成されたフォルダ内の `setup.exe` を実行してインストールします。
3.  アンインストールは Windows の「設定 > アプリ」から `PPTDragDropAddIn` を削除してください。

## 他のPCへの配布方法

本アドインを他のPCにインストールする場合は、**「発行」で出力されたフォルダ全体**を渡す必要があります。

1.  Visual Studio で「発行」した際に出力された、以下のファイル群を一つのフォルダにまとめます。
    - `Application Files` (フォルダ)
    - `setup.exe`
    - `PPTDragDropAddIn.vsto`
2.  このフォルダを **ZIP形式で圧縮** します。
3.  配布先のPCでZIPを解凍し、中にある `setup.exe` を実行してください。

> [!IMPORTANT]
> `setup.exe` 単体ではインストールできません。必ず `Application Files` フォルダなどが一緒にある状態で実行してください。

### 使い方
1.  PowerPoint で動かしたい図形の **名前を `Drag_` で始まる名前に変更** します（例: `Drag_Card1`）。
2.  スライドショーを開始すると、`Drag_` 図形をマウスで自由にドラッグ移動できます。（タッチでの移動は現在無効化されています）
3.  ペンパレットのアイコンでペン描画・消しゴム・スライドナビゲーションが利用できます。
4.  スライドショーを終了すると、図形は自動的に元の位置に戻ります。

### 既知の問題（開発中）
- タッチスワイプによるドラッグを複数回繰り返すと、まれにオーバーレイウィンドウがフリーズすることがある。
  - **対応**: 現在、安定性を優先するためタッチ操作による図形の移動を一時的に無効化しています。マウスドラッグおよびペンパレット操作は正常動作します。
  - 詳細は [memo20260417.md](memo20260417.md) を参照。

## 開発・メンテナンス

### ブランチ管理
本プロジェクトは Git のブランチ機能を活用して、安定版の保護と新機能開発を両立しています。

- **`main`**: 常に安定して動作するコードを保持します。手元の環境でバグが出た場合は、このブランチに切り替えることで即座に復旧できます。
- **`dev`**: 新機能の追加やバグ修正を行うための作業用ブランチです。

**ブランチの切り替え方法:**
```powershell
# 安定版に戻す
git checkout main

# 開発用に戻る
git checkout dev
```
